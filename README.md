
# Project Title

An accessibility exstension to websites.


## Installation

Install my-project with npm

```bash
  npm install negishut
```
    
## Usage/Examples

```javascript
import negishut from 'negishut'

function App() {
    negishut()
    return <Component />
}
```

## NextJs

```typescript
'use client'
import negishut from 'negishut
import {useEffect} from 'react'

function Accessibility(){
    useEffect(()=>{
        negishut()
    },[])
    return (<></>)
}
```
